primeros pasos en git

comandos git
  1. git init

    crea un respositorio vacio

  2. git add .

    guarda cambios en el repositorio

  3. git commit -m " "

    guarda el trabajo terminado "primera version"

  4. git branch -m main

    renombra la rama master por main

Acontinuacion hay que subir este repositorio a la nube con los pasos que indica en la pagina de git.com

  • git remote add origin https: xxx ,

    aqui se crea la version y se sube a la nube, en xxx se coloca la direccion https que nos proporciona git

  • git push -u origin main

    nos proporciona la ubicacion web de la version creada
