primeros pasos en git

comandos git
  1. git init

    crea un respositorio vacio

  2. git add .

    guarda cambios en el repositorio

  3. git commit -m " "

    guarda el trabajo terminado "primera version"

  4. git branch -m main

    renombra la rama master por main

Acontinuacion hay que subir este repositorio a la nube con los pasos que indica en la pagina de git.com

Como trabajar en una version antigua y volver a guardarla en el mismo repositorio

  1. crear nueva carpeta para la nueva version
  2. abrirla en code
  3. abrir git y abrir el repositrio deseado
  4. seleccionar la version en la que se va a trabajar y abrirla
  5. copiarla y pegarla en code

    con RAW - copy raw file

  6. una vez realizada la modificacion en code, volver a git y agregar el archivo desde la pagina con ADD FILE

Otras opciones

Abrir la version deseada y darle a editar en el lugar, descargar con RAW FILE y abrir el archivo con Code para editarlo. Acontinuacion, una vez terminado, copiar el codigo y pegarlo en la version en que se esta trabajando
